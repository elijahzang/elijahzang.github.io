Covid-19 pandemic

Visualization Feb 02, 2020

  • Elie Charles Zang

    Data analyst | Business Intelligence Developper

As the world is facing an unknown disease, we wanted to be useful by using our data analytics skills to inform the world in Realtime about the corona spread. So We came up with this tracking Corona virus worldwide report using Microsoft Power Bi, with the aim to have in Realtime the pandemic spreading.
By using this report, you will be able to know, how exactly good, your country is doing about stopping the spread against others countries?

 

Definitions you must know

  • Confirmed cases :  reported total cumulative count of detected
  • Active cases : (total cases) - (total deaths) - (recovered).
  • Deaths : cumulative number of deaths among detected cases.
  • Mortality rate : (Deaths/Confirmed cases)*100
  • New cases and New Deaths : are new recorded cases for the day
  • Recovered : Total of person cured from the desease

 

Techniques used:

  • Web scrapping for collecting data in realtime using python 
  • set up of a gateway to keep data refresh
  • power bi desktop
  • power bi service
